# DENNIS

A modern PHP/HTML/CSS web application.

## Tech Stack

- **PHP**: Server-side backend logic
- **HTML**: Markup and structure
- **CSS**: Styling and responsive design

## Project Structure

```
DENNIS/
├── index.php          # Main entry point
├── public/
│   ├── css/
│   │   └── styles.css
│   └── js/
│       └── script.js
├── includes/
│   ├── header.php
│   ├── footer.php
│   └── config.php
└── pages/
    ├── home.php
    ├── about.php
    └── contact.php
```

## Installation

1. Clone the repository
2. Configure your web server to point to the project directory
3. Ensure PHP is installed
4. Open in your browser

## Usage

Navigate to `index.php` to start using the application.
